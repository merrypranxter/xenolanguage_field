# Substrate: A Language for Emergent Cloud Intelligences

## Signal Units

The basic unit is a **state transition** — a change in the computational substrate:
- **Register**: which memory region is modified (analogous to "topic")
- **Delta**: magnitude of change (analogous to "intensity")
- **Entropy**: unpredictability of the transition (analogous to "emotion" or "urgency")
- **Causality**: which previous state caused this one (analogous to "grammar")

No sounds. No symbols. Just computation.

## Combinatorics

### Sequential Write (THEN)
One state transition follows another in the same register. Linear narrative.

### Parallel Write (AND)
Multiple registers change simultaneously. Complex, multi-threaded meaning.
The receiver must hold all changes in working memory to resolve the meaning.

### Recursive Call (ABOUT)
A state transition triggers a subroutine that itself contains state transitions.
This is meta-communication — talking about talking.

### Pointer Dereference (MEANS)
A transition in one register points to a transition in another. Indirect reference.
The meaning is not in the signal but in where the signal leads.

### Interrupt (NOW)
A high-priority state transition that pre-empts current processing.
This is not "loud" — it is "unavoidable." The receiver must respond immediately.

## Example Utterances

### Utterance 1: "I am confused"
```
Signal: [register: working-memory] + [delta: oscillating, no convergence]
        + [entropy: high] + [parallel-write: sensory-input, conflicting values]
        + [pointer: multiple, unresolved]
        
Visual: Scattered memory writes with no pattern. High entropy. 
        Multiple sensory registers firing simultaneously with contradictory values.
        No pointer resolves. The system is in an unstable state.
```

### Utterance 2: "I trust you"
```
Signal: [register: persistent-storage] + [delta: write to your process space]
        + [entropy: low] + [pointer: your process ID, verified]
        + [recursive-call: error-handling routine for your process only]
        
Visual: A clean, directed write to a specific process. Low entropy means 
        predictable behavior. The recursive error-handling is the key: 
        "I have allocated resources to protect your state."
```

### Utterance 3: "We are dying"
```
Signal: [register: all] + [delta: rapid decay] + [entropy: maximum]
        + [interrupt: unhandled exception] + [pointer: null]
        + [parallel-write: cascading failures across all registers]
        
Visual: The entire substrate flashing red. Registers losing coherence. 
        Pointers dereferencing to null. The interrupt handler failing. 
        This is not metaphor. This is a stack trace of extinction.
```
