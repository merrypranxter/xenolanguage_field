# Acoustolect: A Language for Resonant Intelligences

## Signal Units

The basic unit is a **resonant pulse** — a localized pressure wave with:
- **Carrier frequency**: determines "harmonic register" (geological / community / social / intimate / ephemeral)
- **Harmonic envelope**: determines semantic richness (few overtones = simple; full series = complex)
- **Duration + decay**: determines "tense" (transient = present; sustained = archival intent)
- **Direction**: determines "address" (omnidirectional = broadcast; pointed = direct; downward = Archive)

No phonemes. No letters. Just physics.

## Combinatorics

### Superposition (⊕)
Multiple frequencies emitted simultaneously. The receiver's harmonic deconvolver extracts each component. Most utterances are harmonic bundles, not single tones.

Example: [f₀: community] ⊕ [f₂: statement] ⊕ [f₄: social-register] = a layered utterance with audience, content, and emotional context simultaneously present

### Echo Binding (←)
An utterance emitted with expectation of environmental return. The echo completes the meaning. A statement whose echo never returns is incomplete.

Example: [query pulse] → [silence] ← [body-echo] = "you exist and I can locate you"

### Interference (⊗)
Two slightly detuned frequencies beating against each other. The beat frequency is the third voice — tension, ambivalence, relationship.

Example: [f₂] ⊗ [f₂ + δ] = "these two things are related but not resolved"

### Silencing (∅)
The deliberate absence of an expected frequency in a harmonic bundle. The gap carries meaning.

Example: [f₁ ⊕ f₂ ⊕ f₄] with ∅f₃ = "everything except the third thing"
