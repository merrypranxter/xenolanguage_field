# Chronolect: A Language for Temporal Swarms

## Signal Units

The basic unit is a **phase packet** — a localized temporal signature with:
- **Carrier frequency**: determines "register" (past-memory, present-action, future-intent)
- **Envelope**: determines "certainty" (sharp = definite, broad = probable/hypothetical)
- **Phase offset**: determines "perspective" (which moment the speaker weights most)

No phonemes. No letters. Just interference packets in time.

## Combinatorics

### Superposition (AND)
Two phase packets occupying the same temporal band interfere. The result encodes both meanings simultaneously. The receiver "hears" both via Fourier decomposition.

Example: [past-memory: sharp] ⊕ [future-intent: broad] = "I remember wanting this" or "I predict I will have wanted this"

### Entrainment (THEN)
One phase packet locks to the frequency of another, pulling it into phase. This creates sequential binding — the closest thing to "syntax."

Example: [present-action] entrained to [past-memory] = action motivated by memory

### Harmonic Cascade (BECAUSE)
A packet at frequency f triggers subharmonics at f/2, f/3, etc. These encode justification, causation, or emotional resonance.
