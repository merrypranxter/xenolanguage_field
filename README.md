# xenolanguage_field

A creative coding project building alternative semiotic systems — languages for non-human intelligences.

## What This Is

Not "alien fonts" or substitution ciphers. This project builds **genuine alternative communication systems** with internal logic, expressive range, and cognitive plausibility. Each language is designed for a specific sensorium, temporal experience, and selfhood model.

The goal: ontologies rendered as signal.

## Project Structure

```
species/          # Species definitions: sensorium, temporality, selfhood
languages/        # Complete language systems: phonology, morphology, syntax
signals/          # GLSL/audio/procedural outputs: the language as raw signal
corpus/           # Example utterances with translation notes
```

## Running

- Visual languages: self-contained GLSL fragment shaders
- Sonic languages: Web Audio API patches, SuperCollider code
- Spatial languages: procedural geometry (Three.js, Blender)
